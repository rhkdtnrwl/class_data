🗃️ STEP 02 · 리스트
리스트 = 칸칸이 상자! 🗃️

리스트는 상자 여러 개를 이어 붙인 것이에요!
각 칸에 번호가 붙어 있어요 😊

🤔 변수 vs 리스트, 뭐가 다를까요?
😓 변수만 쓰면…

⚙️ STEP 03 · 함수
함수 = 마법 버튼! ⚙️

같은 일을 반복할 때,
그 일을 버튼 하나에 담아두는 것이에요!
버튼을 누르면 자동으로 쫘악! 😎

🎰 자판기처럼 생각해봐요!
📥 넣는 것
이름 "민준"
→
⚙️ 함수 (기계)
인사하기( )
→
📤 나오는 것
"안녕! 나는
민준이야!" 👋
재료를 넣으면 → 기계가 처리해서 → 결과가 나와요!
엔트리의 "블록 묶음"이 바로 함수예요! 🎮
🤷 함수가 없으면 어떻게 될까요?
❌ 함수 없이…
😩 민준한테 인사하기
😩 지아한테 인사하기
😩 현우한테 인사하기
😩 서연한테 인사하기
→ 30명이면 30번 써야 해요! 😫
→
✅ 함수 쓰면!
⚙️ 인사하기( ) 버튼 만들기
▶ 민준 인사!
▶ 지아 인사!
▶ 현우 인사!
▶ 서연 인사!
→ 몇 명이든 버튼 하나로 OK! 😎
🔧 함수는 이렇게 만들어요!
1
이름 정하기
"인사하기", "점수계산" 같이 하는 일을 나타내는 이름을 붙여요!
2
할 일 넣기
버튼을 누르면 어떤 일을 할지 써요. 예) "안녕! 나는 ○○이야!" 출력
▶
버튼 누르기! (호출하기)
이름을 부르면 → 자동으로 실행! → 결과가 쫘악 나와요 🎉
🎉 함수의 좋은 점!
🔁 반복 코드가 줄어요
🛠️ 고칠 때 한 곳만!
♻️ 여러 곳에서 재사용!
